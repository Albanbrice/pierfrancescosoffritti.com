Shuffly

Shuffly is meant to help discover new content on YouTube and to watch videos without having to worry about what to choose.

With Shuffly I had the chance to apply all I learned in term of code engineering and UI/UX design since OneCalculator.

It has been a pretty challenging app to develop, because i had to overcome some "limitation" (for my purposes) of the YouTube Data API without at the same time sacrifice the user experience.
I've learned a lot of new cool stuff such as SQLite, the OAuth2 authentication process, event buses (❤) and Guava, thread pools and many more.
The app uses a SQLite database in order to reduce at minimum the number of interactions with the YouTube server. The database is regularly kept consistent by some worker threads.
Having a lot of asynchronous operations, events are a big deal in this app. The use of event buses has made my life a lot easier, allowing for a really awesome level of decoupling.



Shuffly is currently released as alpha on the PlayStore.

HeightMap Generator

A Java heightmap generator based on Perlin Noise and Simplex Noise. It can send data through web socket to a javascript application in order to generate the 3D model of the heightmap. This may not be the most efficient way to send data, but has been useful to play with websockets :)

The maps can be rendered and moved in real time (should be improved) and with different visual effects.
