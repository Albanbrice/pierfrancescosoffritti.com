Shuffly

Shuffly is ment to discover new content on YouTube and to watch videos without having to worry about what to choose.
This time i've been pretty careful following some engieneering principles so the app is pretty extensible an maintainable.
It has been pretty challenging to develop, bacause i had to overcome some 'limitation' in the YouTube Data API without at the same time sacrifice the user experience.
I've learned a lot of new cool stuff such as SQLite, the OAuth2 authentication process, EventBusess (wow) and Guava, ThreadExecutor and many more.
The app uses a SQLite database in order to recude at minimum the number of interactions with the YouTube server. The database is regularly kept consisted by some worker threads.
Having a lot of asynchronous operations, events are a big deal in this app. In a first implementation events where dispatched using interfaces



Shuffly is currently released as alpha on the PlayStore.

HeightMap Generator

A java height map generator based on Perlin Noise and Simplex Noise. It can send data through web socket to a javascript application in order to generate the 3D model of the visible height map. It may not be the best efficent but it's been useful to practise with web sockets :)

The maps can be renderend and moved in real time (should be improved) and with different visual effects.
