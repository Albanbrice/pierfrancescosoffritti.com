Shuffly

Thanks to its pseudo-random algorithms, Shuffly is the best place to discover new content on YouTube or just to chill and spend some time.

Shuffly is not a replacement for the YouTube app, they serve different purposes.

Shuffly transforms YouTube in something similar to the old concept of television, where the user is able to choose the channel, but not exactly what he's going to see.

One Calculator

One Calculator is the only calculator app you need on your Android device :)

The app is capable of solving common calculations, matrix calculations, conversions between units of measurement and number bases and of drawing graphs of functions

One Calculator has been quite successful on the PlayStore and now has more than 100.000 downloads and a rating of 4.6/5


HeightMap Generator

A Java heightmap generator based on Perlin Noise and Simplex Noise.

The app can send data through web socket to a javascript application that is capable of generating the 3D model of the heightmap.

The maps can be rendered and moved in real time and with different visual effects.

RemoteVR

The RemoteVR project is a simple attempt to obtain platform/hardware independent multiplayer Virtual Reality.

The gist of the idea is to run the VR application on a server and then connect many clients to it, allowing them to interact with the VR application in streaming.

Using this approach is easy to have more than one person in the same virtual environment. Also, those people can use different devices and OS to access the same VR experience.
